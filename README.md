**YouTubeSubber - 带双语字幕的YouTube视频下载工具**
这是一个用于下载YouTube视频并自动嵌入精美双语字幕（中文和英文）的工具集。该项目可帮助创建具有教育价值的内容或语言学习材料，通过将双语字幕直接嵌入视频文件中使内容更易于理解。
